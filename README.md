# hexo blog

## 特性（Features）

基于 Hexo [hexo-theme-matery](https://github.com/blinkfox/hexo-theme-matery) 主题改造，进行修改：

- 增加 jsDelivr cdn
- 增加 github action 持续集成
- 修改背景及主题颜色

## 安装使用（Installation）

```shell
git clone https://github.com/realwds/hexo-blog.git
npm install hexo-cli -g
npm install
hexo s
```
